Release Notes

Release Notes

This section provides an overview of all major releases, including new features, improvements, and fixes introduced in each version of CIB ins7ght.

Version 1.0 — Initial Release

Release Date: October 2025

Overview

This marks the first official release of CIB ins7ght, establishing the foundation for process performance and intelligence analytics.
It enables users to connect their CIB seven environments and gain meaningful insights into how their automated processes perform in practice.

Key Features

  • Process Data Extraction:
    Continuous synchronization with CIB seven via REST API, importing data from key tables:
    act_hi_procinst, act_hi_actinst, act_ge_bytearray, act_re_procdef, and act_hi_incident.

  • Global Overview:
    Displays execution trends, process instance counts, and key statistics across all deployed processes.

  • Process Analysis: Bottleneck:
    Drill down into individual processes to identify bottlenecks, inefficiencies, and activity-level performance.

  • Interactive Interface:
    Clean, intuitive dashboards with dynamic filters and hover insights.

Upcoming Version 1.1 — Planned Updates

Status: In Development

Stay tuned for future updates. Each release builds on user feedback and continuous improvement to make CIB ins7ght an even more powerful process intelligence tool.
